4. **変数 `k` の計算**
    ```python
    k = pow(10, len(str(N)), mod)
    ```
    `k` は \( 10 \) を \( N \) の桁数だけべき乗し、その結果を `mod` で割った余りです。`pow` 関数を使用することで効率的に計算しています。
